▷ ul(unordered list) : 리스트 순서 상관 없음, 불릿
-속성값은 3개, circle, disc, square, 속성값은 html5에서는 지원 하지 않음
-레이아웃을 구성 할 때, navigation, board, sitemap 등 목록 형태의 컨텐츠를 표현하는 
중요한 레이아웃 블럭 이라는 점
- > 직계표시
-ul>li*3 = li가 3개 생김, 확정구조

-동시 편집 : ctrl + alt + 방향키↓, 동시 편집 끄기 esc

-숏핸드 : 줄여쓰는 문법

-블럭 특징 : 너비가 따라옴, 100% 중 50%만 써도 아래 내용이 50%를 채우지 못함

▷ol(ordered list) : 리스트 순서 상관 있음, 숫자 목록
-속성값은 4개(+@), "A, a, I, i"
-중요한 것은 레이아웃을 구성 할 때, 번호로 정렬 되어야 하는 목록 형태의 컨텐츠를
표현하는 레이아웃 블럭 이라는  점
-ol>li*5{-----.} = 리스트가 5개 생김, 한글로는 마침표를 꼭 찍어야 됨, 확정구조

▷dl(definition list) : 정의형 목록, li못써, 확정구조
-속성값 없음
-dl>dt>dd
-tittle(dt)을 기준으로 연관 및 보완되는 내용의 data(dd)를 추가 삽입할 때 유리
-dt : 정의할 문자, dl 안에서 무한정으로 쓸 수 있음
-dd : dt를 정의함, dl 안에서 무한정으로 쓸 수 있음
 (ex.<dt>html</dt>
        <dd>hyper text  ----</dd>)
-하나의 dt에 여러개의 dd를 쓰는게 이상적, 갯수가 정해진건 없음
-dl은 dt위로 올라 올 수 없음, dl은 순서 상관없음

(ex. ul>li>dl>dt{모음전}>dd>strong>mark)
